# NEXOS Architecture Viewer — Interactive Viewer

Interactive master-detail viewer of the NEXOS v3.0 architecture. Browse through 46 agents, 6 phases and 9 SOIC dimensions of the autonomous web pipeline.

> Built by [Mark Systems](https://github.com/AILabManager-tech/mark-systems). Companion to the [NEXOS pipeline](https://github.com/AILabManager-tech/nexos-v3).

## Features

- Interactive tree view of the NEXOS architecture
- Master-detail navigation (click to explore)
- Breadcrumb navigation trail
- 46 agents across 6 phases
- 9 SOIC dimensions with weights
- Cyberpunk dark mode design
- Static site — zero dependencies

## Tech Stack

| Layer | Technology |
|-------|------------|
| HTML/CSS/JS | Vanilla (no framework) |
| Font | JetBrains Mono (Google Fonts) |
| Design | Dark mode, SVG logo |
| Deployment | Vercel |

## Getting Started
